# Today Board - H5 + PWA 黑板记录工具

把每天的灵感、待办、心情写进黑板。支持文字、图片，一键导出分享。  
每日 0 点自动擦除黑板（保留置顶便签）。

---

## 项目结构

### 核心文件

- `mobile.html` - 移动端主页面
- `app.js` - 核心逻辑（编辑、存储、导出等）
- `styles-v2.css` - 主样式
- `styles-enhanced.css` - 增强样式
- `sw.js` - PWA Service Worker
- `manifest.webmanifest` - PWA 配置

### 资源目录

- `assets/` - 背景图、UI 素材、PWA 图标
- `fonts/` - 手写体字体（Kalam）

---

## 功能概览

- 文本编辑（富文本、粗体/斜体/下划线/颜色/大小）
- 图片添加（拍照/相册）
- 数据持久化（本地存储）
- 一键导出分享（PNG 图片）
- 置顶便签（保留至次日 0 点后）
